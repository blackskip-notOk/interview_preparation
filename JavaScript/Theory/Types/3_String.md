## **–£–ø—Ä–∞–≤–ª—è—é—â–∏–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –≤ —Å—Ç—Ä–æ–∫–æ–≤—ã—Ö –ª–∏—Ç–µ—Ä–∞–ª–∞—Ö**

–°–∏–º–≤–æ–ª –æ–±—Ä–∞—Ç–Ω–æ–≥–æ —Å–ª—ç—à–∞ `(\)` –∏–º–µ–µ—Ç —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏–µ –≤ JavaScript-—Å—Ç—Ä–æ–∫–∞—Ö. –í–º–µ—Å—Ç–µ —Å —Å–∏–º–≤–æ–ª–∞–º–∏, —Å–ª–µ–¥—É—é—â–∏–º–∏ –∑–∞ –Ω–∏–º, –æ–Ω –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç —Å–∏–º–≤–æ–ª, –Ω–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º—ã–π –≤–Ω—É—Ç—Ä–∏ —Å—Ç—Ä–æ–∫–∏ –¥—Ä—É–≥–∏–º–∏ —Å–ø–æ—Å–æ–±–∞–º–∏ - —ç—Ç–æ —É–ø—Ä–∞–≤–ª—è—é—â–∞—è –ø–æ—Å–ª–µ¬≠–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å (escape sequence).
